# 产品规划

## 产品名称

Computer Organization Study Assistant

## 产品目标

本软件面向《计算机组成原理》期末备考场景，帮助用户完成从知识点学习、OBE 综合任务训练、作答复盘到模拟考核的完整复习流程。

项目同时服务于求职展示，重点体现桌面应用开发、本地数据持久化、学习数据分析和工程化发布能力。

## 用户画像

- 专业：网络工程、计算机科学与技术、软件工程、物联网工程
- 阶段：本科在读
- 基础：计算机组成原理基础薄弱，需要结构化复习
- 目标：通过期末考试，并积累一个可展示项目

## 第一阶段范围

`v0.1 OBE 任务练习版` 只实现必要闭环：

1. 展示章节列表。
2. 展示课程目标。
3. 展示综合任务。
4. 输入主观作答。
5. 对照参考答案。
6. 记录基础练习进度。

## 暂不实现内容

以下功能不进入 `v0.1`：

- 用户登录
- 云同步
- 在线题库
- AI 自动讲题
- 多端同步
- 复杂权限系统
- 选择题、判断题、填空题等小题模式

## 求职展示重点

项目完成后，在简历中突出以下能力：

- 使用 Tauri 构建轻量级 Windows 桌面应用。
- 使用 React 和 TypeScript 开发可维护前端界面。
- 使用 SQLite 管理题库、错题和学习记录。
- 通过数据统计定位薄弱章节。
- 使用 GitHub Release 发布 Windows 安装包。
