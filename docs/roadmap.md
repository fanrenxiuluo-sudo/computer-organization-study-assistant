# 开发路线

## 第 1 步：项目基础骨架

创建 GitHub 展示所需的项目说明、产品规划、数据库设计和基础数据目录。

## 第 2 步：创建前端桌面应用框架

建立 React + TypeScript + Tauri 项目结构，完成首页布局。

## 第 3 步：实现 OBE 任务练习闭环

实现章节选择、课程目标展示、综合任务展示、主观作答、参考答案对照。

## 第 4 步：实现错题本

自动记录错题，支持错题复练和错误次数统计。

## 第 5 步：实现学习统计

统计练习题数、正确率、章节掌握度和每日学习记录。

## 第 6 步：实现模拟考试

支持限时答题、自动评分和结果汇总。

## 第 7 步：打包发布

生成 Windows 可执行文件，补充截图，发布 GitHub Release。
